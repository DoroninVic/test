# Комментарии к реализации
В проекте может быть принят общий подход работы с суммами через копейки. 

Все входящие параметры сумм сразу приводятся к копейкам, в качестве целочисленного значения, что исключает дальнейшие 
проблемы связанные с плавающей точкой, а также необходимость повсеместно контролировать и округлять вычисления до 2 
символов после запятой.
